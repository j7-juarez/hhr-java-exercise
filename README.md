# hhr-java-exercise

Prerequisites
- Java 8 or higer
- Maven 2

Installing
1. Go to project folder: /path/you/extract/heavenhr-interview
2. Pacakge project: mvn package

Run
java -jar api/target/api-1.0.jar

The application will run on port 8080 and contains the api documentation in http://localhost:8080/swagger-ui.html
